Nicolas Cabrera
Checker and evaluator of VRPTR without carpooling solutions
Number of instances: 40
-------------------------------------------------------------------------------------------------------------
If you want to use this code:
1. Check that all the instances files from Coindreau et al. (2019) in txt format are on the "instances" folder.
2. Add all your solutions to the "solutions" folder. Your solutions should have the format: AlgorithmName-instance.txt. 
For example: "MSH-1.txt"
3. Open the class "Main" in the "sensitivity" package.
4. Run as a Java Application.

All your results and reports will be store on the "results" folder.

---------------------------------------------------------------------------------------------------------------

The solutions folder includes the solutions from the MSH and the VNS.

----------------------------------------------------------------------------------------------------------------

The BKS can be established easily with the R code on the folder rcode.

----------------------------------------------------------------------------------------------------------------

For plotting your solutions go to: https://nicolascabrera.shinyapps.io/VRPTR/

----------------------------------------------------------------------------------------------------------

This code is a work in progress. More constraints can be added in the "constraints package".
